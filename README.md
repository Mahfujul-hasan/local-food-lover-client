# Website Name: Local Food Network

## client site live url: https://melodic-tartufo-226969.netlify.app/

## Website features: 
- you can find the updated review for food and resturent
- you can post your reviews on the website
- you can store your review post on my review page
- you can make favorite any review and see them in my favorite page
- you can delete your post
- you can update your post
- you can remove saved review from your my favorite page
- you can login and register in the website.. etc
## Technologies Used
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- TanStack Query

## Core Features
- User authentication and role-based access
- CRUD operations
- Responsive design
- Pagination and optimized data fetching

## Dependencies
- React Hook Form
- SweetAlert
- TanStack Query
- Axios
